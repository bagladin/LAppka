# Учебная аналитика Moodle (LAppka)

Компьютерная система учебной аналитики для поддержки качества тестовых баз Moodle. Предназначена для преподавателей, работающих со стандартными выгрузками Moodle без дополнительных плагинов.

## Возможности

- **Сопоставление данных**: статистика ответов (HTML-отчёт) сопоставляется с экспортом банка вопросов в формате GIFT
- **Категоризация вопросов**: по сложности (лёгкие/средние/сложные), типу (открытые/закрытые), выявление вопросов «на переделку»
- **Коэффициент KBTB**: интегральная метрика сбалансированности тестовой базы с настраиваемыми целевыми долями
- **Экспертная система**: рекомендации по балансу сложности, низкой дискриминации, малому числу попыток; ориентировочное количество вопросов для добавления
- **Person-Item Map**: IRT-визуализация в логит-шкале
- **Экспорт в GIFT**: категоризированный банк с суффиксом (Lappka) для импорта в Moodle

## Формат данных

- **HTML-отчёт** — экспорт результатов тестирования из Moodle (Quiz statistics)
- **GIFT** — экспорт банка вопросов из Moodle

## Установка и запуск

### Требования
- Python 3.8+
- pip

### Установка
```bash
pip install -r requirements.txt
```

### Запуск
```bash
streamlit run app_new.py
```

Приложение доступно по адресу: http://localhost:8501

## Модули

1. **Анализ вопросов** — визуализация тестовых вопросов с метриками и вариантами ответов
2. **IRT Анализ** — Person-Item Map, распределение сложности и способностей
3. **Экспертная система** — KBTB, рекомендации, связанные слайдеры L/M/H (сумма 100%)
4. **Категоризация** — разбиение по категориям, экспорт в GIFT
