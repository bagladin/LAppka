# Документация LAppka

Система учебной аналитики для поддержки качества тестовых баз Moodle.

## Быстрый старт

```bash
# Рекомендуется использовать виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate   # на Windows: .venv\Scripts\activate

pip install -r requirements.txt
streamlit run app_new.py
```

> **Примечание:** Если вы скопировали папку проекта, пересоздайте venv: удалите `.venv` и выполните команды выше заново.

1. Загрузите HTML-отчёт со статистикой и GIFT-файл с вопросами
2. Используйте модули для анализа, категоризации и экспорта

## Структура проекта

```
LAppka/
├── app_new.py           # Главный файл приложения
├── config/              # Конфигурация
├── modules/             # Модули приложения
│   ├── base/            # Парсинг HTML, загрузка данных
│   ├── question_analysis/  # Анализ и визуализация вопросов
│   ├── irt_analysis/    # Person-Item Map (IRT)
│   ├── expert_system/   # KBTB, рекомендации
│   └── categorization/  # Категоризация, экспорт GIFT
├── utils/               # Утилиты
├── static/              # CSS, изображения
├── data/                # Тестовые данные (в т.ч. data/test_files/)
└── docs/                # Документация
```

## Модули

| Модуль | Описание |
|--------|----------|
| **Анализ вопросов** | Визуализация вопросов, метрики, цветовое кодирование по сложности |
| **IRT Анализ** | Person-Item Map, коэффициент перекрытия |
| **Экспертная система** | KBTB, целевые доли L/M/H, рекомендации с ориентировочным количеством вопросов |
| **Категоризация** | Разбиение на категории, экспорт в GIFT с суффиксом (Lappka) |
