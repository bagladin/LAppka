## Модули (актуальные)

| Модуль | Описание |
|--------|----------|
| Анализ вопросов | Визуализация вопросов, метрики |
| IRT Анализ | Person-Item Map |
| Экспертная система | KBTB, рекомендации |
| Категоризация | Категории, экспорт GIFT |

## Источники данных

- **HTML** — экспорт Quiz statistics из Moodle (основной)
- **GIFT** — экспорт банка вопросов (модуль «Категоризация»)
- **CSV** — поддерживается для совместимости
