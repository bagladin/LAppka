# Быстрый старт LAppka

## Установка и запуск

```bash
pip install -r requirements.txt
streamlit run app_new.py
```

## Источники данных

1. **HTML-отчёт** — экспорт результатов тестирования из Moodle (Quiz statistics)
2. **GIFT-файл** — экспорт банка вопросов (загружается в модуле «Категоризация»)

## Модули

| Модуль | Назначение |
|--------|------------|
| **Анализ вопросов** | Визуализация вопросов, метрики (индекс легкости, дискриминации) |
| **IRT Анализ** | Person-Item Map, распределение способностей и сложности |
| **Экспертная система** | KBTB, рекомендации по балансу L/M/H, ориентировочное количество вопросов |
| **Категоризация** | Категории (лёгкие/средние/сложные, открытые/закрытые), экспорт в GIFT |

## Порядок работы

1. Загрузите HTML-файл со статистикой в любом модуле (данные общие)
2. В модуле «Категоризация» загрузите GIFT-файл с вопросами
3. Настройте целевые доли L/M/H в блоке KBTB (модуль «Экспертная система»)
4. Изучите рекомендации и экспортируйте категоризированный банк
